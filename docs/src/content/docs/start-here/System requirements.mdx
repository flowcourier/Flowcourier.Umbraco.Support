## Supported Umbraco versions

The extension is tested on:

- **Umbraco 10.x**
- **Umbraco 11.x**
- **Umbraco 12.x**
- **Umbraco 13.x**

Support for Umbraco 14–17 is in progress — see the [FAQ](/docs/faq/#does-the-extension-support-umbraco-14-17).
