# NuGet Package Installation

The extension is available via NuGet. Visit [Google Analytics by Flowcourier on NuGet](https://www.nuget.org/packages/Flowcourier.Umbraco.Analytics/), search for the package in Visual Studio, or run the following command:

<Code code="dotnet add package Flowcourier.Umbraco.Analytics" lang="shell"  />
